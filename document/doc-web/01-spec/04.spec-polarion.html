1.4.2.任务命名

  1. 所有带有 @ 符号的任务为广播任务，广播任务表示每个人必须执行和完成的任务（自建任务模式）

    属性 默认值

    Type

    Task Tpl

    Categories

    @SELF TASK

    Title

    通常使用@TPL前缀

    Severity

    Must Have

    Priority

    Highest

  2. 任务标题基本命名规范

    • 2.1.只有广播任务使用 @ 符号前缀。

    • 2.2.应用级任务使用各自应用前缀，如 APP / ISO 的任务标题使用 ISO/ 格式。

    • 2.3.任务标题中的数字带单位表示改任务的预估时间。

    • 2.4.个人任务全部使用 PT 前缀，个人任务可不填些任何时间相关信息。

  3. 所有任务最终会有一个期望，表示该任务的完成目标。
